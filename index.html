不忘初心到底是不是一句空话？

发表于 2019-11-08 | 分类于 comment |
字数统计: 2.6k | 阅读时长 ≈ 8

最近参与了不少的主题教育活动，也恰巧在这段时间刷到了一些相关的视频和知乎回答，产生了一些想法想和周围人分享一下。[主要还是因为回想起国庆期间接触的“隐形的守护者”当中的主人公，作为地下党在为国家做出诸多牺牲后仍保持一名党员最初的信仰的那份热诚，一时激动写下了这篇文章。]

提出这个话题源于我在知乎上看到的话题“什么是资本主义？”其实这个问题我在很长一段时间都没能理解，包括这个话题以及衍生话题下的回答都没有很清楚的讲明白。我觉得真的要理解初心，或许从从理解什么是资本主义，什么是社会主义出发比较好。

什么是资本主义？

有相当一部分人认为只要存在商人雇佣劳动力从事工作就是资本主义，因而有人因此而认为我们完全摒弃了原有的道路，已经偷偷修改了自己的姓名。

曾经我也存在这样的迷茫，认为那句 “不管是黑猫还是白猫，能抓老师就是好猫”就是被背离道路的开始。直到自己大学开始静下心来学习马原，才多多少少开始理解“不忘初心”这句话在这个时代具有多大的意义。

资本主义就是在所有生产要素中，以资本为本位，其他生产要素，诸如劳动力，并不占主导地位。所有的生产活动的目的都是为了资本的增殖而服务，至于劳动力只不过不起眼的工具，而至于提供劳动力的人过得怎么样并不是资本需要考虑的事情，除非劳动力已经严重影响了资本本身的发展。所以我们看到在资本主义发展初期那种吃人不吐骨头的丑恶嘴脸，而随后爆发了大量的工人运动不得不让资本做出妥协，承诺给工人提供相应的福利待遇。

乍一看，我们发现似乎工人的生活水平得到了很大的改善，资本主义也已经不再是那个吃人的恶魔。然而，当把视野拉远，从整个时间周期来看，就会发现一件不寻常的事情。由于在资本主义社会一切以资本为本位，大资本吞并小资本，在严格的物权保护制度下， 社会化大生产和生产资料私人占有制之间出现了矛盾。

典型的例子就是由于资本主义过度追求资本增值，因此 衍生出了大量的金融衍生品以保证资本的快速增值。由于我大学期间没有系统学习过金融学的相关知识，细节部分难以做出描述，但是根据教材的讲解和老师的介绍，实行资本主义的国家他们的金融衍生品存在结构复杂，种类繁多的特点，要想分析清楚这些金融衍生品所依赖的资产需要层层穿透。一旦原始的资产出现巨大问题，带来的风险那是难以想象的。相信大家还没有忘记2008年的金融风暴。

另外由于资本主义过度放纵了资本本身的特点，因此资本会直接推动企业不断进行技术创新以保障占有更大的市场份额，从而形成垄断地位以获取更多利润。这里展现出的积极性是资本本身的特点带来的，并不是资本主义带来的。但是一旦出现严重的金融危机，资本本身受到伤害，那么由于奉行资本主义，资本会毫不顾忌其他生产要素，产业升级受阻，失业率高攀，社会发展几近停滞。直到通过诸如航海大发现、西部掘金、海外殖民等手段找到新的经济增长点以度过危机。

说到这里，需要补充一点， 资本并不是资本主义。

之所以这么说，是因为资本本身没有立场，他只是一种生产要素，并不是因为社会上使用了资本进行生产就意味着这个社会奉行的是资本主义。与之相应的，资本家仅是利用资本或者被资本利用的人，而主动与被动的关系由资本家自身的立场决定。

在信奉资本主义的资本家那里，受资本的督促，他会竭尽所能去制定方案以保障获取更多的资本。例如大数据杀熟，就是资本为了不仅从缺钱的人那里赚钱，还要从有钱人那里赚钱，自然想到了低价卖给缺钱的人薄利多销，而凭借用户粘性从有钱人那里卖高价。这么难看的吃相自然会让很多人感到不愉快，从而产生厌烦心理。就不知道未来的价格推荐算法会不会更加智能以至于大家根本察觉不到呢？

那么为什么在我们生活的环境中还能看到各路资本家呢？他们对于我们而言到底是什么样的存在？说实话我曾经也愤青般的认为这些都是离经叛道的人。但是当思考了“什么是社会主义？”这个问题之后，我改变了想法。

“什么是社会主义”

社会主义就是以社会利益为本位，劳动力作为主导生产要素，其他生产要素都必须为劳动力服务。而劳动力背后就是劳动者，凡是不利于劳动者的都是应该做出舍弃的。但是要实现社会主义就必须做到清晰的衡量劳动力的价值。我相信现阶段要做到这一点是十分困难的，因为这个并不是一个容易量化的指标。但凡出现了衡量的不平衡，这势必会造成劳动力的下降，对人的激励将受到十分严重的影响，社会发展也就陷入困境难以自拔。

既然现如今的社会体系做不到完全按劳分配，那么我们就一定要承认其他生产要素在社会发展中的作用以促进生产力的发展也就是提高人们的生产积极性以保障社会的进步。这就是社会主义初级阶段实行的策略。

之前提到资本本身的特点是趋利性，而正是这样的特点让资本能够高效的推动商品经济的发展，商品经济发展势必会促进劳动力的发展。这是资本在自身增值过程中带来的好处，这也是为什么我们如今十分强调资本在社会发展的作用。

但是我们深知放纵资本会是怎样的后果，所以我们制定了很多的监管政策来保障资本尽可能按照服务社会生产力的路子走。

另外，我们对富人以及高收入企业征收了很高的税款，目的就是在于平衡在商品经济竞争中由于某些先天劣势而没能成功的群体，保障他们能够有尊严的在这个社会生活下去。

有键盘侠说这些富人缴了那么多税为什么还是那么有钱，这明显是不对的，应该把他们的财产分一部分给别人。我觉得这样的说法明显就是没有理解社会发展规律，既然我们实行了商品经济，由于资本本身的逐利性，先天会产生贫富差距得趋势，在经济快速发展时期这样的变化尤为明显。这些企业家是在正确的时间做了正确的事，而要做到这一点本身就需要他们自身经历常人难以理解的困难和努力。相反，这些人对于不少人在劳动积极性方面有着一定的影响作用。

从国家的角度看，现如今国际环境十分复杂，这些市场主体在一定程度上对国家经济发展有十分重要的帮助作用，技术创新、劳动力岗位等，相信看看华为就能知道为什么了。还记得历史上我们团结广大农民群众、中产阶级等一切爱国势力一直抵抗外来侵略的成功案例吗？道理是一样的，国家要发展，企业要发展，大家是一条船上的，自然没有树敌一说。当然船舵依然掌握在独立于这些资本家的国家手里。

这也是为什么现如今那么强调广大党员同志尤其是党员干部要做到不忘初心。对，资本带来的优势让人们看到它原来能如此快速得推动社会发展，但是坚持了40年的改革开放，一部分党员放下了对资本的约束，自己不知不觉成为了资本的奴隶，舍弃了基本的底线，真正做到全身心“为资本服务”。

以上就是在前段时间开会了解到的内容，结合自己看到的相关内容总结出的心得体会。作为党员，我觉得自己有义务站出来发声，坚定的表明自己的立场和观点，希望能对周围产生“不忘初心就是空话”的人带来一点启发，也算是一种党员义务。也欢迎大家提出建议，批评指正，理性探讨。

简介

前段时间在翻译外文文献时，我发现很多文献中有很多的数学公式需要插入。我对Latex并不了解，自己一个一个在word中手动输入效率太低，但放截图又感觉太low，所以就想找一种方法能够快速的将数学公式插入到word当中。

需要使用的软件

主要用到两款软件，一个用来识别截图转化为Latex公式，一个用来转化Latex代码方便插入到Word当中。因为我是白嫖党，所以用到的软件全部不需要付费或者破解。

  • Mathpix Snip Tool
  • 任何一个在线编辑Markdown的编辑器。例如Cmd Markdown，也可以是RStudio中Rmarkdown文件生成的网页，重要的是需要有Show Math as MathML Code的功能。

Mathpix Snip Tool的使用

打开Mathpix Snip Tool这个网站，找到适合自己电脑系统的版本下载下来安装。

他默认的截图快捷键是Ctrl+Alt+M，这个快捷键可能会与一些PDF阅读器的快捷键冲突，所以要在设置中改为不冲突的快捷键，我设置的是Ctrl+Alt+F。

每次你截完图他就会自动识别出图片中的数学公式，然后复制下面的Latex公式。emm，我也不知道每一行具体的含义，我用的是第二行的那个。

转化为MathML Code

把复制的Latex公式复制到Markdown在线编辑器中，他会自动渲染出我们期望的数学公式。

右键点击右边的数学公式，选择 Show Math as——MathML Code，然后就会跳出一个小框，复制里面的全部内容，然后以纯文本形式粘贴到word中，这样公式就出现了。

plm包简介

面板数据是计量经济学中一个不断发展的领域。随着单位横断面(如家庭、公司、国家等)上所观察到的数据越来越多，以及随着时间的推移，出现了一些利用这种双维性来处理一些与经济数据有关的典型问题的估计方法，首先就是未观察到的异质性问题。

长期以来，对来自不同观测单元的数据进行时间上的观察在其他统计领域(通常称为纵向数据)很常见。在面板数据领域和其他领域，计量经济学方法在实验环境方面是独特的，因为它强调模型说明和测试，并处理与经济数据有关的特定统计问题所引起的一些问题。

因此,虽然一个非常全面的软件框架(在许多其他特性)：最大似然估计线性回归模型的纵向数据,包nlme和lme4,可在R环境中使用。

例如,使用随机效应估计面板模型,对于一名正在练习的计量经济学学生而言，使用它并不直观。而最大似然估计只是面板数据计量经济学的一种可能方法。此外，经济面板数据集往往是不平衡的，这一情况需要对方法进行一些调整，与nlme中的方法并不兼容。

因此，需要一个“从计量经济学家的角度”进行面板数据处理的软件包，并且至少具备计量经济学家所使用的基本技术:静态线性面板数据模型的随机和固定效应估计、可变系数模型、动态模型的广义矩估计方法;以及规范和错误规范诊断的基本工具箱。

此外，我们认为有必要自动化一些基本的数据管理任务，例如滞后、求和，更一般地说，对数据应用(R意义上的)函数，这些函数虽然概念简单，但在二维数据上却很麻烦，容易出错，特别是在不平衡面板的情况下。

数据结构

面板数据具有特殊的结构:每一行数据对应一个特定的个体和时间段。在plm中，数据参数可以是一个普通的data.frame，但在本例中，必须添加一个名为index的参数来指示数据的结构。这可以是:

  • NULL假设前两列包含个体和时间索引，观测值按样本个体名称和时间周期排序;
  • 一个字符串，应该是单个索引的名称;
  • 长度为2的字符向量，包含单独的名字和时间索引;
  • 一个代表样本个体数量的整数(只能在按照样本个体名称排序的平衡面板的情况下使用)。

然后在内部调用pdata.frame函数，该函数返回一个pdata.frame，这是一个具有index属性的data.frame。这个属性是一个data.frame，它包含样本个体名称和时间索引。样子大概就是下面这样。

界面

估计接口(Estimation interface)

plm提供各种估计功能，其中:

  • plm:基本面板模型的估计，即组内、组间和随机效应模型。利用lm函数对转换后的数据进行模型估计;
  • pvcm:变系数模型的估计;
  • pgmm:广义矩模型法的估计;
  • pggls:一般可行广义最小二乘模型的估计;
  • pmg:对于非均匀面板模型，均值组(MG)、降低的MG (DMG)和共同相关效应MG (CCEMG)的估计量;
  • pcce:有共同因素的面板数据的共同相关效应平均组(CCEMG)和合并组(CCEP)的估计量;
  • pldv:有限因变量的面板估计。

这些函数的接口与lm()函数一致。也就是说，它们的前两个参数是公式和数据(应该是data.frame，并且是强制性的)。这些函数还有三个常见的参数:

  • index:这个参数使估计函数能够识别数据的结构，即个体和每次观测的时间周期;
  • effect:模型中要包括的效应，即个体效应、时间效应或两者兼有的效应;
  • model:要估计的模型的种类，大多数时候是具有固定效应的模型或具有随机效果的模型。

这四个函数的结果存储在一个对象中，这个对象的类具有与函数相同的名称。它们都继承自类panelmodel。panelmodel对象包含:coefficients, residuals, fitted.values, vcov, df.residual 和 call,另外还提供了提取这些元素的函数。

测试接口(Testing interface)

诊断测试接口为大多数函数提供了公式和panelmodel方法，但也有一些例外。因此，用户可以选择测试先前估计的panelmodel对象中的结果，或者为了测试而重新估计它。

尽管第一种策略是最有效的，但对面板模型的诊断测试主要使用来自pooling模型对象的OLS残差，这些残差的估计在计算上并不费事。

检验

正如在第2节中概述的，面板模型中的规范测试主要涉及到对可汇集性、个体或未观察到的时间效应以及这些效应与回归量之间的相关性的测试(Hausman-type tests)。对于其他常见的诊断检查，我们提供了一套序列相关测试，但没有涉及异方差测试的问题。相反，我们提供了异方差鲁棒协方差估计器，将在6.7小节中描述。

混合性检验(Tests of poolability)

pooltest检验假设，即相同的系数适用于每个个体。这是一个标准的F检验，它是基于对整个样本的模型和基于对每个个体的方程估计的模型的比较。

pooltest的第一个参数是plm对象。第二个参数是使用model="within"获得的pvcm对象。如果第一个参数是pooling模型，则测试适用于所有系数(包括截距项)，如果它是模型中的，则假设不同的截距项。

为了检验Grunfeld例子中的所有系数(不包括截距)都相等的假设，我们使用:

F statistic
data: inv ~ value + capital
F = 5.7805, df1 = 18, df2 = 170, p-value = 1.219e-10
alternative hypothesis: unstability

当然，使用pooltest()函数时，我们可以只使用公式就可以得到同样的结果。

独立性和时效检验

可以添加另外两个参数来指示要计算的测试类型。参数类型为:

  • "honda": Honda (1985), the default value,
  • "bp": Breusch and Pagan (1980),
  • "kw": King and Wu (1997),
  • "ghm": Gourieroux, Holly, and Monfort (1982)

测试的效果用"effect"参数表示(“individual”、“time”或“twoways”中的一个)。所实现的测试统计数据也适用于不平衡面板。

R语言面板数据分析

发表于 2019-03-18 | 分类于 R |
字数统计: 859 | 阅读时长 ≈ 2

横截面数据

指多个观测对象在某一时间点上的数据，例如2018年中国各省GDP。

面板数据

指多个观测值在不同时间点上的取值。比如：1994-2018年中国各省的GDP。

为什么有人说面板数据是纵向数据特殊的形式？因为纵向数据他对各个对象不同的观测值不一定按照相同的时间点记录，各个对象的观测次数和观测间隔都不一定一样。

分析纵向数据模型都有一个基本点，那就是 把不同的个体的差异作为随机效应加入回归模型之中以用来反映个体差异对重复测量的结果的影响 。这些随机效应描述了每个对象的度量随时间的变化，并解释了纵向数据的相关结构。

面板数据分析当中往往还有固定效应的部分，固定效应可以理解为类似通常的线性模型。

什么是房地产投资？

先来看一下官方教材上的定义：

经济主体以获得未来的房地产资产增值或收益为目的，预先垫付一定数量的货币与实物，直接或间接地从事或参与房地产开发与经营活动地经济行为。

从定义上看，我们可以明确两点：

  • 投资的目的：获取增值或者收益
  • 投资的方式：直接或间接

从投资目的上看，我们可以把投资方式分为通过转售获取增值部分收益，或者可以通过经营这一房产获得其收益。当然，这二者也并不是完全独立的，还应该从实际的业态形式来进行区分。

而从投资方式上看，直接投资房地产是指直接参与房地产开发或购买房地产的过程并参与有关管理工作，包括了从购地开始的开发投资和物业建成之后的置业投资两种。间接投资是指将资金投入与房地产相关的权益或证券市场的投资活动。

为什么会出现城市？

发表于 2019-03-12 | 分类于 reading notes |
字数统计: 1.9k | 阅读时长 ≈ 6

1 城市形成前的地区——庭院生产

“庭院生产”是一个我们一开始提出来的模型，并不是实际存在的，主要是根据这个模型一步一步放松限定条件和假设来说明不同类型城市出现的原因。

庭院生产模型有以下几个限定条件和假设：

  • 我们为了方便，假设人们只需要生产和消费面包和衬衣
  • 不存在集中地生产或者集中地人口，每个消费者都是生产者，他们都在自家庭院生产
  • 相同的生产率，不管地上种什么庄稼其生产率都相同，不管工人生产什么产品，其生产率都相同
  • 交易的规模收益不变，无论双方交易多少，单位产品的交易成本都是固定的。
  • 生产的规模收益不变，无论工人能够生产多少产品，他每小时生产的数量都是固定的。

那么这些假设会带来什么样的结果呢？很明显，这些庭院生产者们并不会进行交易行为。并不存在因为我专业化生产面包或者衬衣就能带来额外收益，因为 生产的规模收益不变 意味着我扩大生产并不会导致我生产的总量增加， 交易的规模收益不变 意味着我扩大交易量，并不会导致我的交易成本(例如交通成本)会降低， 相同的生产率 则意味着我们生产速度都一样，并不存在我能多生产多少某一产品从而导致自己产量增加。

土壤矿物质

土壤矿物质是土壤的“骨架”，一般能够占到土壤固相部分重量的95%-98%。固相的其余部分为有机质、土壤微生物体，所占比例小，占固相重量的5%以下。

之所以我们需要关注土壤矿物质，是因为 土壤矿物质的组成、结构和性质会对

  • 土壤物理性质(结构性、水分性质、通气性、热学性质、力学性质和耕性)
  • 化学性质(吸附性能、表面性能、酸碱性、氧化还原电位、缓冲作用等)
  • 生物与生物化学性质(生物多样性、酶活性等)

都有深刻影响。
