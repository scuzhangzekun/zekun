创建一个散点图

ggplot2中，图是用“+”串起来的，每个函数都有属于自己的一部分。下面就创建一个最基础的散点图。

创建一个简单的散点图

我们来分解这个作图步骤：

  • ggplot()函数初始化了图形并且指定了数据源为mtcars，aes()函数则是代表aesthetics，即使用视觉效果呈现信息，我们在这里指定了x轴的信息是wt(汽车重量)，y轴的信息是mpg(每米所消耗的油量);
  • geom_point()表面了我们希望绘制散点图的想法；
  • labs()是说明我们需要在图上增加注释，标题、x、y轴标签。

创建一个带有线性拟合的散点图

我们发现我们增加了一些内容。

  • geom_point(pch=17,color=”blue”,size=2)中，pch指定了点的形状，color指定了点的填充颜色，size指定了点的大小，能够明显感觉到点变大了。
  • geom_smooth(method=”lm”,color=”red”,linetype=2)则是增加了一条平滑的曲线，我们使用的method是线性拟合(method=”lm”)，并且color=”red”指定了颜色为红色，linetype=2指定了线型为虚线。值得注意的是，默认情况下，这条平滑的曲线是包括在95%置信区间(暗带)中的。关于线性拟合的内容会在以后分享。

创建一个分组的散点图

ggplot2为我们提供了分组和小面化的方法。分组是指在一个图形中显示多组数据观察结果，小面化则是值在单独、并排的图形是显示观察组。

注意： ggplot2在定义组或者面时使用的是因子型数据。

我们可以使用mtcars中的am、vs、cyl变量来进行分组。

从图中我们看到，每个点的颜色和形状代表了不同的汽车发动机气缸数量，而不同的分组和面则表示了变速箱类型（自动还是手动），发动机装置（V型还是直列式）

常见的几何函数指定图形类型

ggplot()需要我们指定绘制的数据源和变量，几何函数则是制定我们需要绘制图表的类型。常见的图表有以下这些：

函数 添加 选项
geom_bar() 条形图 color,fill,alpha
geom_boxplot() 箱线图 color,fill,alpha,notch,width
geom_density() 密度图 color,fill,alpha,linetype
geom_histogram() 直方图 color,fill,alpha,linetype,binwidth
geom_hline() 水平线 color,alpha,,linetype,size
geom_jitter() 抖动点 color,alpha,size,shape
geom_line() 线图 color,alpha,linetype,size
geom_point() 点图 color,fill,alpha
geom_rug() 地毯图 color,alpha,shape,size
geom_smooth() 拟合曲线 method,formula,color,fill,linetype,size
geom_text() 文字注解 参见“帮助”
geom_violin() 小提琴图 color,fill,alpha,linetype
geom_vline() 垂线 color,alpha,linetype,size

对于这些函数中的选项，其实也是有章可循的。

参数 解释
color 对点、线、面的轮廓进行着色
fill 对面的内部区域进行着色
alpha 颜色的透明度，0是完全透明，1是不透明
linetype 图案线条(1=实线，2=虚线，3=点，4=点破折号，5=长破折号，6=双破折号)
size 尺寸大小
shape 点的形状(0=开放的长方形，1=开放的圆形，2=开饭的三角形)
position 对条形图来说，“dodge”将分组条形图并排显示，“stacked”堆叠分组条形图，“fill”垂直堆叠分组条形图并规范其高度一致。对于点来说，“jitter”减少点的重叠
binwidth 直方图的宽度
notch 表示方块图是否有缺口（true/false）
sides 地毯图的安置(“b”=底部，”l”=左边，”t”=上部，”r”=右部，”bl”=左下部)
width 箱线图的宽度
